Images the Angular Way

Using a binding with the img src attribute results in a 404 error in the console window but the image still prints to screen


    <img src={{country.flag}}" alt="">

Using the ng-src fixes the 404 error


    <img ng-src="{{myimage.jpg}}" alt="">

Two-way Data Binding

To bind data two-ways, use an ngModel along with the curly braces

{{ twowaymsg }}
