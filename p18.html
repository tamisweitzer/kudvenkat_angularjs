Using the $http Service

The $http service is used to make http requests to a server.

Using this service you can make a GET, POST, PUT, or DELETE request to a database on a server.


      let app = angular.module('app', []);
    app.controller('appCtrl', function($scope) {
      
      $http.get({
        method: 'GET',
        url: 'urlToServer'
      })
      .then( function(response) {
        // handle response.data
      });
    });
